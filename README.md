# moovy
React Interactive UI Components

## Preview the UI components library

`packages/ui/.storybook` is the Storybook config directory for the UI component library.

The default UI primitive system for this package is now **Radix UI** (`radix-ui`).

From the repository root, run:

```bash
npm install
npm run storybook -w @moovy/ui
```

Then open:

```text
http://localhost:6006
```

If you prefer running from the package directory:

```bash
cd packages/ui
npm run storybook
```

Press `Ctrl + C` in the terminal to stop the preview server.
